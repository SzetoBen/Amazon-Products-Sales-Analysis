# Amazon-Products-Sales-Analysis
DS 2003 Final Project

Dataset: https://www.kaggle.com/datasets/ikramshah512/amazon-products-sales-dataset-42k-items-2025?resource=download

# How to Setup Repository

1. Navgiate to root directory `cd Amazon-Products-Sales-Analyis`
2. Create virtual enviornment  `python -m venv venv`
3. Activate virtual enviornment `.\venv\Scripts\activate`
4. Install python libraries in virtual enviornment `pip install -r requirements.txt`
5. Unzip the `dataset/amazon-dataset.zip` file within the dataset folder
6. Run scripts within the scripts folder
